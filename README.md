# Pet Boarding App

## Project Documentation

This project is designed to provide a platform for pet boarding services, connecting pet owners with boarding facilities.

### Features
- User authentication
- Search for pet boarding facilities
- Bookings management
- User reviews for boarding facilities

## Setup Instructions

### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nihalmurtaza/pet-boarding-app.git
   cd pet-boarding-app
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application
To start the application, run:
```bash
npm start
```

### Testing
To execute the tests, run:
```bash
npm test
```
